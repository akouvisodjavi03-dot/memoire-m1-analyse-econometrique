# Mémoire de validation M1 – Analyse économétrique sous R

## Contexte académique
Mémoire réalisé dans le cadre de la validation du Master 1.
Le travail porte sur une analyse économétrique appliquée à l’aide du logiciel R des déterminants d'accès aux moyens de préventions antipaludéens en Afrique subsaharienne.

## Objectif du mémoire
Analyser empiriquement une problématique économique à partir de données quantitatives
en mobilisant des méthodes économétriques adaptées.

## Méthodologie
- Nettoyage et préparation des données
- Analyse statistique descriptive
- Estimation de modèles économétriques
- Interprétation des résultats

## Outils
- R
- Packages standards d’analyse de données et d’économétrie

## Contenu du repository
- Scripts R utilisés pour l’analyse
- Fichiers de travail (hors données confidentielles)

## Remarque
Les données originales et le mémoire complet ne sont pas publiés
pour des raisons académiques et de confidentialité.
